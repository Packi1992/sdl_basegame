

                How to use this example


Use Buttons to Select Tileset (NO TILESET / TILESET 1 - 3)

Press "SHOW TILESET" and Select Tile by leftclick

Press Save Button to save map



ESC                     -> Close Tileset Selection / Hint

strg R + L Click on Map -> place selected Tile

n                       -> show Tilenumbers

g                       -> show Tilegrid


                      Player Controls

                a                -> move left

                d                -> move right

                strg L           -> roll

                space            -> jump
